# 📝 LAB 1 — AI for Lesson Planning

## Duration
15 Minutes

## Objective
Faculty create:
- Lesson plan
- CO mapping
- Classroom activity
- Bloom’s Taxonomy alignment

## Demonstration Flow
- Manual lesson planning → 2 hours
- AI lesson planning → 2 minutes

### Prompt 1 — Basic Lesson Plan
```text
Act as an experienced university faculty member.

Create a 60-minute lesson plan for the topic:
“Machine Learning Basics”

Include:
1. Learning objectives
2. Introduction activity
3. Core concepts
4. Real-world examples
5. Classroom interaction
6. Assessment activity
7. Conclusion

Use Bloom’s Taxonomy levels.
```

### Prompt 2 — Outcome-Based Education
```text
Generate Course Outcomes (COs) and map them with Bloom’s Taxonomy for the subject:
“Data Structures”

Create:
- 5 COs
- Cognitive levels
- Practical activities
```

## Hands-on Task
Participants should:
- Choose their own subject
- Generate lesson plan
- Modify difficulty level
- Create activity for slow learners

## Duration

## Objective

- MCQs
- Case studies
- Rubrics
- Quiz questions

### Prompt 1 — MCQ Generator
```text
Generate 10 university-level MCQs on “Cyber Security”.

Requirements:

- Add explanation for each answer
- Difficulty level: Moderate
```

```text

“Software Engineering”
```

### Prompt 3 — Rubric Generator
```text
Create an assessment rubric for evaluating student presentations.

Criteria:
- Content quality
- Communication
- Innovation
- Technical understanding
- Team collaboration

Use a 5-point scale.
```

# 🏆 Golden Framework: R-C-A-F Prompt Model

| Component | Meaning |
|---|---|
| R | Role |
| C | Context |
| A | Action |
| F | Format |

### Example
- **Role:** “Act as a professor…”
- **Context:** “For MBA students…”
- **Action:** “Create lesson plan…”
- **Format:** “Use table format…”
