# 📚 LAB 1 — AI for Lesson Planning

## 🎯 Objective

In this lab, faculty members will learn how to leverage AI tools for effective lesson planning and Outcome-Based Education (OBE).

By the end of this lab, participants will be able to create:

- 📝 Lesson Plans
- 🎯 Course Outcome (CO) Mapping
- 🤝 Classroom Activities
- 🧠 Bloom’s Taxonomy Alignment

---

# 🚀 Demonstration Flow

## Traditional vs AI-Based Lesson Planning

### ⏳ Manual Lesson Planning
- Time Required: **~2 Hours**

### ⚡ AI-Assisted Lesson Planning
- Time Required: **~2 Minutes**

### 🎓 Learning Outcome

Participants will observe how AI significantly reduces planning effort while improving instructional design quality.

# 🧩 Prompt 1 — Basic Lesson Plan

## 🎯 Purpose

Generate a structured lesson plan with Bloom's Taxonomy alignment.

### 💬 Prompt

```text
Act as an experienced university faculty member.

Create a 60-minute lesson plan for the topic:
“Machine Learning Basics”

Include:
1. Learning objectives
2. Introduction activity
3. Core concepts
4. Real-world examples
5. Classroom interaction
6. Assessment activity
7. Conclusion

Use Bloom’s Taxonomy levels.
```

### 📌 Expected Output

- Clearly defined learning objectives
- Bloom's Taxonomy integration
- Classroom engagement activities
- Assessment strategy
- Lesson conclusion
