# 🎓 RCAF Prompt Framework for Higher Education

## 📌 What is RCAF?

**RCAF** is a simple prompt engineering framework designed for educators to create effective AI prompts quickly and consistently.

| Letter | Component | Description |
|----------|------------|-------------|
| **R** | 👤 **Role** | Who should the AI act as? |
| **C** | 📚 **Context** | Background information and target audience |
| **A** | ⚡ **Action** | The specific task to perform |
| **F** | 📄 **Format** | How the output should be structured |

### ✅ Why Use RCAF?

- Easy for faculty to remember
- Improves prompt clarity
- Generates more accurate AI responses
- Applicable across teaching, assessment, and academic planning
- Reduces prompt-writing effort

# 🚀 RCAF Prompt Templates for Higher Education

## 1️⃣ Lesson Planning

### 🎯 Template

#### 👤 Role
Act as an experienced university professor in **[Subject]**.

#### 📚 Context
The audience is **[Year/Semester]** students with basic knowledge of **[Topic]**.

#### ⚡ Action
Create a lesson plan for a **[Duration]-minute** session on **[Topic]**.

#### 📄 Format
Present the lesson plan in a table with the following columns:

- Time
- Topic
- Learning Objective
- Teaching Method
- Activity

### 💡 Example Prompt

> Act as an experienced Data Structures professor. For second-year engineering students, create a 60-minute lesson plan on Linked Lists. Present the output in a table with Time, Topic, Learning Objective, Teaching Method, and Activity.

---

## 2️⃣ Question Paper Generation

### 🎯 Template

#### 👤 Role
Act as an experienced university examiner.

#### 📚 Context
The course is **[Course Name]** for **[Year/Semester]** students. The syllabus covers **[Topics]**.

#### ⚡ Action
Generate a question paper with a mix of:

- Easy questions
- Medium questions
- Difficult questions

#### 📄 Format

Present questions:

- Section-wise
- With marks allocation
- With Bloom’s Taxonomy level

### 💡 Example Prompt

> Act as an experienced university examiner. The course is Database Management Systems for third-year engineering students. Generate a 50-mark question paper covering SQL, Normalization, Transactions, and Indexing. Present it section-wise with marks and Bloom's Taxonomy levels.

## 4️⃣ Rubric Design

### 🎯 Template

#### 👤 Role
Act as an academic assessment expert.

#### 📚 Context
The rubric will be used to evaluate **[Assignment / Project / Presentation]**.

#### ⚡ Action
Create a grading rubric with:

- Clear performance criteria
- Multiple score levels
- Transparent assessment standards

#### 📄 Format

Present the rubric in a table containing:

| Criteria | Excellent | Good | Satisfactory | Needs Improvement | Marks |
|-----------|------------|--------|--------------|-------------------|--------|

### 💡 Example Prompt

> Act as an academic assessment expert. Create a rubric for evaluating student project presentations in Artificial Intelligence. Present the rubric in a table with performance levels and marks distribution.

### 🎯 Template

#### 👤 Role
Act as an experienced faculty member.

#### 📚 Context
The learners are **[Student Level]** studying **[Topic]**.

#### ⚡ Action
Generate a quiz that assesses:

- Conceptual understanding
- Application skills
- Critical thinking

#### 📄 Format

Provide:

- Multiple-Choice Questions (MCQs)
- Four options per question
- Correct answers
- Brief explanations

### 💡 Example Prompt

# 📝 Summary 

| Task | 👤 Role | 📚 Context | ⚡ Action | 📄 Format |
|--------|----------|------------|-----------|------------|
| **Lesson Plan** | University Professor | Student level & topic | Create lesson plan | Table with objectives & activities |
| **Question Paper** | Examiner | Course & syllabus | Generate exam paper | Sections, marks & Bloom's level |
| **Assignment** | Course Instructor | Student knowledge level | Create assignment | Objectives, questions & deliverables |
| **Rubric** | Assessment Expert | Assignment/Project type | Design rubric | Performance criteria table |
| **Quiz** | Faculty Member | Topic & learner level | Generate quiz | MCQs with answers & explanations |

---

# 🎯 Key Takeaway

> **RCAF = Role + Context + Action + Format**

By following this simple framework, educators can create **clear, structured, and effective AI prompts** for lesson planning, assessments, assignments, rubrics, and quizzes, leading to higher-quality educational content and improved teaching efficiency. 🚀📚✨
