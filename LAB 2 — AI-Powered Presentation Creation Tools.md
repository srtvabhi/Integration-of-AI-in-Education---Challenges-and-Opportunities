# 🎨 LAB 2 — AI-Powered Presentation Creation Tools

## 🎯 Objective

In this lab, participants will learn how to use AI-powered presentation tools to create professional, engaging, and educational presentations with minimal effort.

By the end of this lab, participants will be able to:

- 📊 Generate presentations using AI
- 🎓 Create subject-specific educational content
- 🎨 Design visually appealing slides automatically
- ⚡ Reduce presentation creation time significantly
- 🤖 Utilize multiple AI presentation platforms effectively

# 🛠️ Tool 1 — Gamma AI

## 🌐 Platform

🔗 https://gamma.app/

---

## 🎯 Purpose

Generate a complete educational presentation automatically from a single prompt.

### 💬 Prompt

```text
Create a comprehensive and visually engaging presentation for B.Tech 2nd Year Computer Science Engineering students on the topic "Binary Search Trees (BST) in Data Structures".

Audience:
Undergraduate engineering students who have basic knowledge of programming in C/C++ and fundamental data structures.

Presentation Goals:
• Explain Binary Search Trees from basic to intermediate level.
• Build conceptual understanding through visual examples.
• Cover theoretical concepts, algorithms, code implementations, and applications.
• Prepare students for university examinations, coding interviews, and practical programming.

Presentation Length:
5–8 slides.
```

---

## 📌 Expected Output

### 📖 Content Coverage

- Introduction to BST
- BST Properties
- Tree Traversal Methods
- Search Operation
- Insertion Algorithm
- Deletion Algorithm
- Code Implementation
- Real-world Applications

### 🎨 Design Features

- AI-generated layouts
- Smart visuals
- Professional themes
- Interactive slide design

---

## 🎯 Purpose
